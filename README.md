## Usage

First, make sure you have Node >= 8 and NPM >= 5.

``` bash
# install Quasar CLI:
$ npm install -g @quasar/cli

# generate a project folder
$ quasar create <folder_name> --kit github:bildvitta/asteroid-starter-kit
```
